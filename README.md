## 기능

#### 홈 화면
***
+ 스플래시
+ 캐릭터 선택
+ 캐릭터 생성 및 삭제 (이름, 직업, 능력치)
+ 종료

#### 메인 화면
***
1. 상태보기
2. 던전입장
3. 인벤토리
4. 상점
5. 장비창
x. 나가기

#### 1. 상태창
***
+ 캐릭터 정보
+ 나가기

#### 2. 던전입장
***
+ 조우 
  + 전투정보(몬스터)
  + 전투 / 다른 곳 둘러보기 / 도망
+ 전투화면
  + 전투정보(플레이어/몬스터)
+ 전투종료화면
  + 현재 상태
  + 획득한 아이템
  + 계속 전투 / 마을로

#### 3. 인벤토리
***
+ 아이템 목록
+ 소유한 골드
+ 나가기

#### 4. 상점
***
+ 장비 구매 및 판매
+ 강화
+ 소유한 골드
+ 나가기

#### 5. 장비창
***
+ 각 아이템 유형으로 구별된 장비창
+ 가지고 있는 장비
+ 나가기
